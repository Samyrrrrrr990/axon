# Axon v1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build Axon v1 ‚Äî a local-first visual (node-graph) platform for training ML models, fine-tuning pretrained models, and composing LLM/agent pipelines ‚Äî then publish to GitHub with launch/growth collateral.

**Architecture:** Python-native monorepo. One FastAPI server exposes REST + WebSocket APIs and serves the built React frontend. A pure-Python execution engine runs JSON node graphs (topological order, content-addressed disk caching, live event streaming). Nodes are decorated Python functions whose UI is auto-generated from their declaration. Heavy ML dependencies install lazily per "pack."

**Tech Stack:** Python 3.12 (uv-managed venv), FastAPI, uvicorn, pydantic v2, pandas, scikit-learn, httpx, SQLite (stdlib sqlite3); lazy packs: xgboost, torch, transformers+peft+datasets, chromadb+sentence-transformers. Frontend: React 18 + TypeScript + Vite + @xyflow/react (React Flow 12) + zustand + Tailwind CSS v4 + Recharts.

## Global Constraints

- Python version pinned to **3.12** via `.python-version` (torch/transformers compatibility; system 3.14 is too new for the ML stack).
- Package name / CLI command: **`axon`**. Server port: **8700**.
- License: **PolyForm Noncommercial 1.0.0** in `LICENSE.md`, plus `COMMERCIAL_LICENSE.md`.
- Workspace dir: `~/.axon/` (override with env `AXON_HOME`) ‚Äî config, cache, runs DB, workflows.
- Workflow file format: single-file `.axon.json`, no absolute paths inside, `"format": "axon-workflow/1"`.
- Core install must stay light: **no torch/transformers/chromadb in core deps**. Lazy packs only.
- LLM-calling nodes are **never cached** (`cacheable=False`); all trainable nodes accept a `seed` param.
- Copilot default provider: **OpenRouter free model** (`openrouter`, model default `deepseek/deepseek-chat-v3.1:free`); Anthropic / OpenAI / Ollama configurable.
- All copilot output is **graph ops JSON**, never executable code.
- Frontend: dark-first, distinctive visual identity. When building UI invoke `frontend-design:frontend-design`; before any chart component invoke `dataviz` skill.
- Every commit message ends with `Co-Authored-By: Claude Fable 5 <noreply@anthropic.com>`.

### Task 5: Engine ‚Äî executor, caching, run manager, CLI

**Files:**
- Create: `axon/engine/cache.py`, `axon/engine/events.py`, `axon/engine/executor.py`, `axon/engine/runs.py`, `axon/cli.py`
- Test: `tests/engine/test_cache.py`, `tests/engine/test_executor.py`

**Interfaces:**
- `events.py`: `RunEvent(type: str, run_id: str, node_id: str|None = None, data: dict = {})`; types: `run_started, node_started, node_log, node_progress, node_finished, node_failed, run_finished, run_failed, run_cancelled, pack_install_progress`.
- `cache.py`: `Cache(dir: Path)` ‚Äî `key(node_type, params, input_keys: list[str], pack_version: str) -> str` (sha256 of canonical JSON), `.get(key) -> dict|None` (unpickle `{outputs}`; corrupt/missing ‚Üí None), `.put(key, outputs)`, `.clear()`. Pickling failure (unpicklable model handles) ‚Üí skip caching silently, log once.
- `executor.py`: `execute_workflow(wf: Workflow, registry, cache, workspace: Path, settings: dict, emit: Callable[[RunEvent], None], cancel: threading.Event, run_id: str) -> dict[node_id, dict[socket, container]]`. Synchronous (called in a worker thread). Per node in topo order: cancelled? ‚Üí `run_cancelled` and stop; compute cache key (LLM/`cacheable=False` nodes and any node downstream of one get fresh keys via a `uuid` salt); cache hit ‚Üí emit `node_finished` with `{"cached": true, "preview": ...}`; else build `NodeContext` wired to emit, call fn, normalize outputs (bare container ‚Üí only output socket; wrong dict keys ‚Üí `node_failed`), emit `node_finished` with previews `{socket: container.preview()}`. Exceptions ‚Üí `node_failed` with `{"error": str, "traceback": last 3 frames, "hint": _hint_for(exc)}` (`_hint_for` maps `ModuleNotFoundError` ‚Üí "install pack X", common shape/dtype errors ‚Üí plain-language hints), skip all downstream-dependent nodes (emit `node_failed` with `"skipped": true`), continue independent branches, then `run_finished` (status `"error"` if any failure).
- `runs.py`: `RunManager(registry, cache, workspace, settings_getter)` ‚Äî `.start(wf) -> run_id` (spawns `threading.Thread`), `.cancel(run_id)`, `.subscribe(cb)` / `.unsubscribe(cb)` (fan-out every event; also appended to an in-memory ring buffer per run for late GET), `.get_run(run_id) -> dict` (status, per-node statuses/previews, timings).
- `cli.py`: `main()` argparse ‚Äî `axon start [--port 8700] [--no-browser]` (imports server, uvicorn.run, `webbrowser.open` after startup); `axon run <file.axon.json>` (headless: load registry, validate, execute, print per-node status lines, metrics of terminal nodes, exit 1 on failure ‚Äî this is the e2e-CI entry point).
